## Export Environment Variable
```
# Your own AWS Access Key ID
$ export AWS_ACCESS_KEY_ID='xxxxxxxxxxxxx'

# Your own AWS Secret Access Key
$ export AWS_SECRET_ACCESS_KEY='xxxxxxxxxxxxxxxxxx'
```
